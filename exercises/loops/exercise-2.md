# Exercise 2 - Count Down Timer

**Task Description:**

Write a loop that counts down from 30 to 0, printing each number to the console. After reaching 0, it should print "Time's up!".

**Objective:**

This exercise is designed to reinforce your understanding of loop mechanics in JavaScript, focusing on implementing a simple countdown timer.

**Example Output:**

```plaintext
30
29
...
1
0
Time's up!
```

**Instructions:**

1. Use a for loop that starts at 30 and decrements down to 0.
2. Print each number to the console as the loop iterates.
3. Once the loop completes, print "Time's up!" to the console.
